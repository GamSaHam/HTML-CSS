함수 선언 및 호출

선언함수 : 자주 사용되는 명령들(기능)을 하나의 이름으로 선언하여 필요한 경우 호출하여 사용

=> 함수를 호출하여 함수의 명령을 지속적으로 사용

		function 함수명(매개변수,...) {
			명령;
			...
		}

함수 호출 => 함수명(값,...);

익명함수 : 명령들의 모임

=> 함수를 호출하여 함수의 명령을 한 번만 사용(이벤트 처리)

=> 익명함수를 변수에 저장하여 호출 가능

		var 함수변수명=function (매개변수,...) {
			명령;
			...
		}

함수 호출 => 함수변수명(값,...);
