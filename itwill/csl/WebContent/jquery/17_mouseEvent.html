마우스 이벤트

click : 마우스 버튼을 누른 경우

dblclick : 마우스 버튼을 두번 연속 누른 경우

mouseenter : 이벤트 엘리먼트 영역에 마우스 커서가 위치한 경우

mouseleave : 이벤트 엘리먼트 영역에 마우스 커서가 위치하지 않은 경우

mouseover : 이벤트 엘리먼트 영역에 마우스 커서가

mouseout : 이벤트 엘리먼트 영역에 마우스 커서가 벗어난 경우

hover : 이벤트 엘리먼트 영역에 마우스 커서가 진입하거나 벗어난 경우
